# 需求理解阶段知识

## 1. 项目定位

### 1.1 产品定义

桌面端本地文件管理工具，为个人用户提供**文件目录管理、文件操作、文件预览、在线Markdown编写**的一体化解决方案。

### 1.2 核心价值

| 价值 | 说明 |
|------|------|
| 一体化 | 目录、文件、预览、MD编辑全功能集成 |
| 离线可用 | 单机模式，无需网络即可使用 |
| 便捷部署 | 双击exe即可运行，无需复杂配置 |
| 数据安全 | 本地加密存储、操作日志追踪 |

### 1.3 目标用户

- 个人用户：需要在本地管理各类文档、资料的普通用户
- 作家/编辑：需要Markdown写作环境的用户
- 学生/教师：需要整理学习资料的用户

## 2. 核心功能模块

### 2.1 功能总览

| 模块 | 核心功能 |
|------|---------|
| 目录管理 | 创建、编辑、删除、移动、查询、面包屑导航 |
| 文件管理 | 上传、编辑、删除、移动、下载、搜索、筛选 |
| 文件预览 | 文档预览、图片预览、音频/视频播放 |
| MD编辑 | 创建、编辑、实时预览、自动保存、导出、分享 |
| 回收站 | 查看、恢复、清空、30天自动清理 |

### 2.2 模块详细说明

#### 目录管理
- **创建**: 名称1-50字符，禁止特殊字符 `/ \ : * ? " < > |`
- **编辑**: 支持重命名、移动，需记录操作日志
- **删除**: 移入回收站，30天后自动永久删除
- **查询**: 支持搜索、面包屑导航、排序

#### 文件管理
- **上传**: 单个≤100MB，批量≤50个/次，MD5去重
- **支持格式**:
  - 文档: doc, docx, xls, xlsx, ppt, pptx, pdf, txt, md
  - 图片: jpg, jpeg, png, gif, bmp
  - 音频: mp3, wav
  - 视频: mp4, avi
- **删除**: 移入回收站，30天后自动清除
- **下载**: 单个/批量ZIP下载

#### 文件预览
- **文档**: PDF/TXT/MD直接预览，Office转换后预览（≤20MB，≤5秒）
- **图片**: 放大、缩小、旋转、切换
- **音频/视频**: 播放、暂停、音量、进度

#### MD编辑
- **格式**: 分屏编辑、单屏编辑
- **功能**: 标题、加粗、斜体、代码块、列表、表格、链接、图片
- **保存**: 30秒自动保存、手动保存
- **导出**: .md / .pdf / .html

#### 回收站
- **恢复**: 单个/批量恢复至原目录
- **清理**: 手动清空、30天自动删除

## 3. 技术架构

### 3.1 技术栈

| 层级 | 技术 | 版本 |
|------|------|------|
| 前端 | Vue 3 + Element Plus | ≥3.0 / ≥2.5 |
| 后端 | Spring Boot + Java | 2.7.x / 17+ |
| 数据库 | SQLite | 3.40+ |
| 构建 | Maven + Vite | 3.9+ / 5+ |
| 打包 | Launch4j | - |

### 3.2 分层架构

```
Controller → Service → Repository → Model
```

| 层级 | 职责 | 禁止操作 |
|------|------|----------|
| Controller | 参数校验 + 响应格式化 | 禁止执行业务逻辑 |
| Service | 业务逻辑核心，事务边界 | 禁止直接访问数据库 |
| Repository | 数据访问层，封装SQL | 禁止执行业务逻辑 |
| Model | 数据模型定义 | 禁止业务逻辑 |

### 3.3 核心模块

| 模块 | Service类 | 核心方法 |
|------|----------|----------|
| 目录管理 | DirectoryService | create/update/delete/getChildren/restore |
| 文件管理 | FileService | upload/download/delete/search/restore |
| 回收站 | TrashService | list/restore/clear/cleanExpired |
| 预览 | PreviewService | getPreviewUrl/convert |
| 存储 | StorageService | save/load/delete |

## 4. 数据模型

### 4.1 核心表

| 表名 | 说明 | 关键字段 |
|------|------|---------|
| directories | 目录表 | id, name, parent_id, path, deleted, delete_at |
| files | 文件表 | id, name, ext, size, md5, path, parent_id, deleted, delete_at |
| md_contents | MD内容表 | file_id, content, updated_at |
| operation_logs | 操作日志表 | id, type, content, created_at |

### 4.2 字段规范

| 类型 | 命名规则 | 示例 |
|------|----------|------|
| 主键 | id | id |
| 外键 | entity_id | user_id, parent_id |
| 名称 | name | name |
| 时间 | *_at | created_at, updated_at, delete_at |
| 软删除 | deleted | deleted (0/1) |

### 4.3 状态机

```
普通目录/文件 → 删除 → 回收站 → 恢复 → 普通状态
                                    ↘ 永久删除（30天后自动/手动清空）
```

## 5. 非功能需求

### 5.1 性能指标

| 指标 | 要求 |
|------|------|
| 启动时间 | ≤10秒 |
| 页面加载 | ≤2秒 |
| 列表加载 | ≤1秒 |
| 10MB上传 | ≤10秒 |
| 文件转换 | ≤5秒（≤20MB） |

### 5.2 安全要求

| 类别 | 要求 |
|------|------|
| 二次确认 | 删除、替换、批量操作需确认 |
| 操作日志 | 记录操作时间、类型、内容，保留90天 |
| 文件安全 | 禁止上传可执行文件，路径穿越防护 |
| 数据备份 | 每日增量，每周全量，保留30天 |

### 5.3 兼容性

| 项目 | 要求 |
|------|------|
| 浏览器 | Chrome/Firefox/Edge/Safari 最新版及前两版 |
| 分辨率 | 1366×768及以上 |
| 操作系统 | Windows 10/11 (32位/64位) |

## 6. 本阶段执行标准

### 6.1 输入要求

| 输入项 | 来源 | 说明 |
|--------|------|------|
| Issue 描述 | docs/issues/issue-{N}-*.md | 功能需求和验收条件 |
| 项目上下文 | CLAUDE.md | 架构约束和技术栈 |
| BRD | docs/PRD/BRD.md | 业务需求（参考） |
| PRD | docs/PRD/PRD.md | 产品需求（参考） |

### 6.2 执行步骤

1. **读取 Issue** - 理解功能需求、验收条件、Boundaries
2. **提取关键信息** - 核心功能、技术约束、依赖关系
3. **明确 Boundaries** - 范围之内（实现）、范围之外（不实现）
4. **识别 Open Questions** - 需要确认的问题

### 6.3 输出要求

| 输出项 | 内容 |
|--------|------|
| Task | 完整的需求理解描述 |
| Boundaries | 明确的范围之内/之外列表 |
| Open Questions | 待确认问题清单 |

### 6.4 验收条件

| 条件 | 判定标准 |
|------|----------|
| 需求理解完整 | 所有验收条件已理解，无遗漏 |
| Boundaries 明确 | 范围之内/之外清晰，无模糊地带 |
| Open Questions 识别 | 关键问题已列出，不影响阶段推进 |
| 信息充分性 | design 阶段可基于此开始技术决策 |

## 7. 关键概念

### 7.1 业务概念

| 概念 | 定义 |
|------|------|
| 目录 | 用于组织文件的层级结构，支持嵌套创建/移动/删除 |
| 文件 | 存储在目录中的具体文档，支持上传/下载/预览 |
| 回收站 | 删除文件临时存储，30天后自动清除 |
| MD文件 | Markdown格式文本，支持实时预览/导出 |
| 预览转换 | Office文档转换为PDF后预览 |

### 7.2 技术概念

| 概念 | 定义 |
|------|------|
| 软删除 | 使用 deleted 字段，删除时标记为1而非物理删除 |
| FTS5 | SQLite 全文搜索索引，用于高效搜索 |
| 路径穿越 | 防止通过 `../` 访问非授权目录 |
| 参数化查询 | 使用占位符防止 SQL 注入 |
