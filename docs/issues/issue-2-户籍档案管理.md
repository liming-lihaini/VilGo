# ISSUE-2-户籍档案管理

## 概述

以家庭户为单位进行管理，支持家庭户信息录入、编辑；可从村民人事档案模块同步获取户主信息，无需重复录入；支持添加、删除、修改家庭成员（关联人事档案，确保信息一致）；记录家庭年度收入台账、户籍变动（迁入/迁出/新生儿/分户/合户），生成变动记录留痕；支持按家庭户编号、户主姓名查询，自动统计家庭户数、总人口数。

## 功能需求

### F-HOU-001 创建家庭户

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 创建家庭户 |
| 需求描述 | 选择户主创建家庭户，系统自动同步户主信息 |
| 输入 | 户主ID（关联已存在的村民档案） |
| 校验 | 户主存在且未被其他家庭户关联 |
| 输出 | 家庭户编号自动生成，档案入库 |
| 异常处理 | 户主已关联其他家庭户提示"该人员已是户主" |

### F-HOU-002 编辑家庭户

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 编辑家庭户 |
| 需求描述 | 修改家庭户信息（家庭户编号不可修改） |
| 输入 | 家庭户ID、要修改的字段及新值 |
| 校验 | 家庭户存在 |
| 输出 | 编辑成功提示，信息更新 |

### F-HOU-003 删除家庭户

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 删除家庭户 |
| 需求描述 | 软删除家庭户，先解除成员关联 |
| 输入 | 删除确认、家庭户ID |
| 校验 | 家庭户存在 |
| 输出 | 确认弹窗，成功后从列表移除 |
| 异常处理 | 有关联成员时提示"请先解除成员关联" |

### F-HOU-004 家庭成员管理

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 家庭成员管理 |
| 需求描述 | 添加、删除、修改家庭成员，与人事档案关联 |
| 输入 | 家庭户ID、家庭成员（村民ID、与户主关系） |
| 校验 | 成员村民档案存在，同一家庭户内关系唯一 |
| 输出 | 成员列表更新 |
| 异常处理 | 成员已关联其他家庭户提示"该人员已是其他家庭户成员" |

### F-HOU-005 年度收入台账

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 年度收入台账 |
| 需求描述 | 记录家庭年度收入信息 |
| 输入 | 家庭户ID、年份、总收入、人均收入、收入来源 |
| 校验 | 年份格式、家庭户存在 |
| 输出 | 收入记录入库 |

### F-HOU-006 户籍变动记录

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 户籍变动记录 |
| 需求描述 | 记录户籍变动类型、时间、原因，生成留痕 |
| 输入 | 家庭户ID、变动类型、变动时间、变动原因 |
| 校验 | 家庭户存在，变动类型有效 |
| 输出 | 变动记录入库 |
| 日志记录 | 操作人、操作时间、变动前后状态 |

### F-HOU-007 户籍查询统计

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 户籍查询统计 |
| 需求描述 | 按家庭户编号、户主姓名查询，统计户数、人口数 |
| 输入 | 查询条件 |
| 校验 | - |
| 输出 | 符合条件的家庭户列表，统计结果 |

### 业务规则

1. 家庭户编号：全局唯一，自动生成（如H2024001）
2. 户主信息从村民档案同步，不可重复录入
3. 家庭成员关联人事档案，确保信息一致
4. 户籍变动生成完整留痕记录
5. 变动类型：迁入/迁出/新生儿/分户/合户
6. 软删除时需先解除成员关联
