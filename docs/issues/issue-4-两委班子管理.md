# ISSUE-4-两委班子管理

## 概述

管理村两委成员基础信息、工作分工（关联对应管理模块）；支持工作任务分配、进度跟踪、完成情况登记，实现任务闭环管理；记录两委会议信息（会议时间、地点、参会人员、会议决议、落实情况），决议事项可关联任务分配；支持班子成员信息查询、分工及任务统计，提升工作透明度。

## 功能需求

### F-TC-001 两委成员管理

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 两委成员管理 |
| 需求描述 | 管理两委成员基础信息（姓名、职务、联系方式等） |
| 输入 | 姓名、性别、职务、联系方式、分管工作、入职时间 |
| 校验 | 姓名非空，职务有效 |
| 输出 | 成员信息入库 |
| 异常处理 | 姓名重复提示"已存在同名成员" |

### F-TC-002 编辑成员信息

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 编辑成员信息 |
| 需求描述 | 修改两委成员信息 |
| 输入 | 成员ID、要修改的字段及新值 |
| 校验 | 成员存在 |
| 输出 | 编辑成功提示，信息更新 |

### F-TC-003 删除成员

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 删除成员 |
| 需求描述 | 软删除两委成员，保留历史任务关联 |
| 输入 | 删除确认、成员ID |
| 校验 | 成员存在，无进行中的任务 |
| 输出 | 确认弹窗，成功后从列表移除 |
| 异常处理 | 有进行中任务提示"请先完成或转移任务" |

### F-TC-004 任务分配

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 任务分配 |
| 需求描述 | 分配工作任务给两委成员，设置完成期限 |
| 输入 | 任务内容、接收人、分配时间、完成期限 |
| 校验 | 接收人存在，期限在当前时间之后 |
| 输出 | 任务入库，状态为"待处理" |
| 日志记录 | 分配人、操作时间 |

### F-TC-005 任务进度跟踪

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 任务进度跟踪 |
| 需求描述 | 更新任务状态（待处理/进行中/已完成），记录成效 |
| 输入 | 任务ID、新状态、完成成效 |
| 校验 | 任务存在，状态流转合理 |
| 输出 | 任务状态更新 |

### F-TC-006 会议记录

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 会议记录 |
| 需求描述 | 记录会议信息及决议事项 |
| 输入 | 会议时间、地点、参会人员、会议内容、决议事项 |
| 校验 | 会议时间格式正确 |
| 输出 | 会议记录入库 |

### F-TC-007 决议关联任务

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 决议关联任务 |
| 需求描述 | 将决议事项转化为任务分配给两委成员 |
| 输入 | 会议ID、决议事项、接收人、完成期限 |
| 校验 | 会议记录存在，接收人存在 |
| 输出 | 任务创建成功 |

### F-TC-008 班子统计

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 班子统计 |
| 需求描述 | 统计成员分工、任务完成情况 |
| 输入 | 统计条件 |
| 校验 | - |
| 输出 | 统计结果（成员数、任务数、完成任务数） |

### 业务规则

1. 成员职务：书记/主任/副主任/委员
2. 任务状态：待处理/进行中/已完成
3. 状态流转：待处理 → 进行中 → 已完成（不可逆）
4. 会议决议可关联创建任务
5. 任务到期前提醒
