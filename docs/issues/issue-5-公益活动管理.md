# ISSUE-5-公益活动管理

## 概述

支持发起村级公益活动（填写活动名称、时间、地点、内容、报名要求）；村民可报名参与（村级工作人员协助录入报名信息）；记录活动出勤情况、记工时长，支持出勤统计；活动结束后可进行归档，保存活动照片、总结等相关资料；支持按活动时间、活动类型统计参与人数、出勤时长，解决人工记工繁琐、统计不便的痛点。

## 功能需求

### F-PAC-001 发起活动

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 发起活动 |
| 需求描述 | 工作人员发起公益活动，填写基本信息 |
| 输入 | 活动名称、活动时间、活动地点、活动内容、报名要求 |
| 校验 | 活动名称非空，时间格式正确 |
| 输出 | 活动创建成功，状态为"未开始" |
| 异常处理 | 活动时间冲突提示"该时段已有其他活动" |

### F-PAC-002 编辑活动

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 编辑活动 |
| 需求描述 | 修改活动信息（未开始的活动中） |
| 输入 | 活动ID、要修改的字段及新值 |
| 校验 | 活动存在且状态为未开始 |
| 输出 | 编辑成功提示，信息更新 |

### F-PAC-003 删除活动

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 删除活动 |
| 需求描述 | 软删除活动（未开始的活动中） |
| 输入 | 删除确认、活动ID |
| 校验 | 活动存在且状态为未开始 |
| 输出 | 确认弹窗，成功后从列表移除 |

### F-PAC-004 村民报名

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 村民报名 |
| 需求描述 | 工作人员协助村民报名参与活动 |
| 输入 | 活动ID、村民ID（关联人事档案） |
| 校验 | 活动存在且未开始，村民档案存在 |
| 输出 | 报名成功，记录报名时间 |
| 异常处理 | 重复报名提示"该村民已报名" |

### F-PAC-005 出勤签到

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 出勤签到 |
| 需求描述 | 记录活动出勤情况（出勤时长、工值） |
| 输入 | 报名ID、出勤时长、工值 |
| 校验 | 报名记录存在 |
| 输出 | 签到成功，状态更新为"已签到" |

### F-PAC-006 活动归档

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 活动归档 |
| 需求描述 | 活动结束后归档，保存照片、总结 |
| 输入 | 活动ID、活动总结 |
| 校验 | 活动存在且进行中 |
| 输出 | 归档成功，状态更新为"已归档" |
| 日志记录 | 归档人、操作时间 |

### F-PAC-007 活动统计

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 活动统计 |
| 需求描述 | 统计参与人数、出勤时长、工值汇总 |
| 输入 | 统计条件（时间范围/活动类型） |
| 校验 | - |
| 输出 | 统计结果（活动数、参与人数、总出勤时长） |

### 数据结构

**公益活动表 (public_activities)**

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK, AUTO | 活动ID |
| name | VARCHAR(100) | NOT NULL | 活动名称 |
| activity_time | DATETIME | | 活动时间 |
| location | VARCHAR(200) | | 活动地点 |
| content | TEXT | | 活动内容 |
| requirements | VARCHAR(200) | | 报名要求 |
| status | VARCHAR(20) | DEFAULT '未开始' | 活动状态 |
| summary | TEXT | | 活动总结 |
| creator | VARCHAR(50) | | 录入人 |
| created_at | DATETIME | | 创建时间 |
| updated_at | DATETIME | | 更新时间 |

**活动报名表 (activity_signups)**

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK, AUTO | 报名ID |
| activity_id | BIGINT | FK->public_activities | 活动ID |
| resident_id | BIGINT | FK->residents | 村民ID |
| sign_time | DATETIME | | 报名时间 |
| work_hours | DECIMAL(5,2) | | 出勤时长 |
| work_value | DECIMAL(10,2) | | 工值 |
| status | VARCHAR(20) | DEFAULT '已报名' | 状态 |

### 业务规则

1. 活动状态：未开始/进行中/已归档
2. 报名状态：已报名/已签到/已取消
3. 出勤时长以小时为单位
4. 工值以元为单位
5. 活动归档后不可再报名
