# ISSUE-文件管理

## 概述

实现文件管理模块的完整功能，包括文件上传、编辑、删除、查询、下载等操作。

## 功能需求

### F-FILE-001 文件上传

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 文件上传 |
| 需求描述 | 支持单个/批量文件上传至指定目录 |
| 输入 | 文件、目标目录（可选，默认当前目录） |
| 约束 | 单个文件≤100MB，批量≤50个/次 |
| 支持格式 | 文档类：doc、docx、xls、xlsx、ppt、pptx、pdf、txt、md<br>图片类：jpg、jpeg、png、gif、bmp<br>音频类：mp3、wav<br>视频类：mp4、avi |
| 输出 | 上传进度（百分比）、成功提示 |
| 唯一性校验 | MD5值相同则拒绝上传 |
| 拖拽上传 | 支持拖拽文件至指定目录上传 |
| 续传支持 | 暂停、继续上传 |
| 失败处理 | 提示失败原因，支持重新上传 |
| 元数据记录 | 文件名、大小、格式、创建人、上传时间、存储路径、MD5值 |

### F-FILE-002 文件编辑

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 编辑文件 |
| 需求描述 | 修改文件名、移动位置、替换内容 |
| 输入 | 新名称、目标目录、新文件（替换时） |
| 校验 | 名称唯一性、目录存在 |
| 输出 | 编辑成功提示 |
| 替换确认 | 替换前需确认提示 |
| 日志记录 | 操作人、操作时间、操作类型、文件信息 |

### F-FILE-003 文件删除

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 删除文件 |
| 需求描述 | 删除文件移入回收站 |
| 输入 | 删除确认 |
| 确认提示 | "是否确认删除此文件？删除后将进入回收站，30天后自动永久删除" |
| 批量删除 | 支持批量删除，需确认提示 |
| 回收站规则 | 保留30天，可恢复可清空 |
| 手动清空 | 支持一键清空（清空后不可恢复） |

### F-FILE-004 文件查询与筛选

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 文件查询与筛选 |
| 需求描述 | 支持文件名搜索、类型/时间/大小筛选 |
| 输入 | 搜索词、筛选条件 |
| 筛选维度 | 文件类型、上传时间、文件大小、创建人 |
| 组合筛选 | 多条件可组合使用 |
| 排序 | 名称、时间、大小升序/降序 |
| 显示信息 | 文件名称、大小、格式、上传时间、创建人 |

### F-FILE-005 文件下载

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 文件下载 |
| 需求描述 | 支持单个下载、批量ZIP下载 |
| 输入 | 文件选择 |
| 输出 | 下载进度、本地保存 |
| 批量格式 | ZIP包，名称格式"文件批量下载_时间戳" |
| 失败处理 | 提示失败原因，支持重新下载 |

#### 关键组件

| 组件 | 描述 | 事件 |
|------|------|------|
| `FileUploader` | 文件上传组件，支持拖拽/点击上传 | `upload`, `progress`, `complete`, `error` |
| `FileList` | 文件列表，支持列表/网格视图切换 | `select`, `preview`, `download` |
| `UploadProgress` | 上传进度条组件 | `cancel`, `retry` |
| `BatchActionBar` | 批量操作工具栏 | `download`, `delete`, `move` |
| `FilterPanel` | 筛选面板（类型/时间/大小） | `change` |
| `DragDropZone` | 拖拽上传区域 | `drop`, `dragover` |

#### 交互流程

1. **上传文件流程**:
   ```
   选择文件/拖拽文件 → 显示进度条 → 计算MD5 → 检查重复
   → 上传中(百分比) → 上传完成 → 刷新列表 → 成功提示
   ```

2. **批量删除流程**:
   ```
   选中多个文件 → 点击"删除" → 确认对话框 → 点击确认
   → 调用DELETE /api/file/delete → 刷新列表 → 移入回收站
   ```

3. **拖拽上传流程**:
   ```
   拖拽文件到窗口 → 高亮提示区 → 松开鼠标 → 开始上传
   → 显示进度 → 上传完成 → 列表更新
   ```

#### 异常提示

| 场景 | 提示信息 |
|------|----------|
| 文件超过100MB | "文件大小不能超过100MB，请选择其他文件" |
| 格式不支持 | "不支持此文件格式，仅支持: doc/docx/xls/..." |
| MD5重复 | "该文件已存在同名文件，内容相同，无需重复上传" |
| 上传失败 | "上传失败，请检查网络后重试" |
| 选中超过50个 | "批量操作最多支持50个文件" |

### 数据结构

**文件表 (t_file)**

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK, AUTO_INCREMENT | 文件ID |
| name | VARCHAR(50) | NOT NULL | 文件名 |
| ext | VARCHAR(20) | - | 扩展名 |
| size | BIGINT | NOT NULL | 文件大小 |
| md5 | VARCHAR(32) | NOT NULL | MD5校验值 |
| storage_path | VARCHAR(500) | NOT NULL | 存储路径 |
| parent_id | BIGINT | FK→t_directory(id) | 所属目录 |
| user_id | BIGINT | FK→t_user(id) | 用户ID |
| created_at | DATETIME | NOT NULL | 上传时间 |
| updated_at | DATETIME | NOT NULL | 更新时间 |
| deleted | TINYINT | DEFAULT 0 | 逻辑删除 |
| delete_at | DATETIME | - | 删除时间(回收站) |

### 索引

```sql
INDEX idx_parent_id (parent_id)
INDEX idx_md5_parent (md5, parent_id)  -- MD5去重
```

### 业务规则

1. 单个文件≤100MB，批量≤50个/次
2. 支持格式：doc、docx、xls、xlsx、ppt、pptx、pdf、txt、md、jpg、jpeg、png、gif、bmp、mp3、wav、mp4、avi
3. MD5相同则拒绝上传（同一目录下）
4. 删除文件移入回收站，保留30天

## 验收条件

1. 上传≤100MB成功 → 进度显示 → 成功提示
2. MD5相同拒绝 → 提示"文件已存在"
3. 删除文件 → 弹窗确认 → 移入回收站
4. 批量下载 → 生成ZIP → 自动下载
5. 搜索文件 → 实时返回结果
6. 文件筛选 → 多条件组合正常
7. 拖拽上传 → 支持拖拽区上传
