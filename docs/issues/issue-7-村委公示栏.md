# ISSUE-7-村委公示栏

## 概述

村级工作人员可发布村级通知、政策文件、财务公开、项目公示、惠民信息等内容，支持图文编辑、修改、删除；公示内容按发布时间排序，村民可查看所有公示内容，支持公示内容查询（按标题、发布时间）；支持村民反馈意见提交与回复。

## 功能需求

### F-NOT-001 发布公示

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 发布公示 |
| 需求描述 | 发布公示内容，支持图文编辑 |
| 输入 | 公示标题、公示内容（支持图片）、公示类型、公示期限 |
| 校验 | 标题非空，内容非空，期限格式正确 |
| 输出 | 公示发布成功 |
| 异常处理 | 标题重复提示"已存在同名公示" |

### 数据结构

**公示栏表 (notices)**

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK, AUTO | 公示ID |
| title | VARCHAR(100) | NOT NULL | 公示标题 |
| content | TEXT | | 公示内容 |
| notice_type | VARCHAR(20) | | 公示类型 |
| publisher | VARCHAR(50) | | 发布人 |
| expire_date | DATE | | 公示期限 |
| status | VARCHAR(20) | DEFAULT '未过期' | 公示状态 |
| created_at | DATETIME | | 创建时间 |
| updated_at | DATETIME | | 更新时间 |

**公示反馈表 (notice_feedback)**

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK, AUTO | 反馈ID |

| resident_id | BIGINT | FK->residents | 反馈人ID |
| content | TEXT | | 反馈内容 |
| reply | TEXT | | 回复内容 |
| created_at | DATETIME | | 创建时间 |

### 业务规则

1. 公示类型：通知/政策文件/财务公开/项目公示/惠民信息
2. 公示状态：未过期/已过期（自动更新）
3. 内容支持富文本（图片插入）
4. 反馈后记录村民ID（关联人事档案）
5. 标题全局唯一性校验
