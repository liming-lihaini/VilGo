# ISSUE-6-党务管理

## 概述

管理党员基础信息（可从人事档案模块关联，补充党员专属信息）；记录党务活动（主题党日、组织生活会等）、规范党员发展流程（入党积极分子、发展对象、预备党员、正式党员各阶段登记）；支持党员信息查询、党费收缴统计、党务活动归档，实现党务工作数字化、留痕化。

## 功能需求

### F-PAW-001 党员档案管理

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 党员档案管理 |
| 需求描述 | 管理党员信息，关联人事档案获取基础信息 |
| 输入 | 身份证号（关联人事档案）、入党时间、转正时间、党员状态 |
| 校验 | 身份证号在人事档案中存在，状态有效 |
| 输出 | 党员档案入库 |
| 异常处理 | 身份证号不在人事档案中提示"人员信息不存在" |

### F-PAW-002 编辑党员档案

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 编辑党员档案 |
| 需求描述 | 修改党员信息（入党时间、转正时间、党员状态） |
| 输入 | 档案ID、要修改的字段及新值 |
| 校验 | 档案存在 |
| 输出 | 编辑成功提示，信息更新 |

### F-PAW-003 删除党员档案

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 删除党员档案 |
| 需求描述 | 软删除党员档案，保留历史记录 |

| 校验 | 档案存在 |
| 输出 | 确认弹窗，成功后从列表移除 |

### F-PAW-004 党员发展

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 党员发展 |
| 需求描述 | 记录党员发展阶段（积极分子→发展对象→预备党员→正式党员） |

| 校验 | 状态流转合理 |
| 输出 | 状态更新 |

### F-PAW-005 党务活动

| 需求项 | 内容 |

### 业务规则

1. 党员状态：积极分子/发展对象/预备党员/正式党员
2. 状态流转：积极分子 → 发展对象 → 预备党员 → 正式党员（顺序不可逆）
3. 入党时间必须早于或等于转正时间
4. 党费按月收缴
5. 身份证号唯一性校验
