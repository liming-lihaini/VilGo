# ISSUE-MD编辑

## 概述

实现在线Markdown文件创建、编辑、保存、导出功能。

## 功能需求

### F-MD-001 MD文件创建

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 创建MD文件 |
| 需求描述 | 在指定目录下创建MD文件并进入编辑 |
| 输入 | 文件名（1-50字符，后缀.md） |
| 同名校验 | 同一目录下唯一 |
| 输出 | 自动进入编辑界面 |
| 默认视图 | 分屏模式（编辑区+预览区） |

### F-MD-002 MD编辑功能

| 需求项 | 内容 |
|--------|------|
| 需求名称 | MD编辑 |
| 需求描述 | 支持分屏编辑、实时预览 |
| 编辑模式 | 分屏编辑（左右）、单屏编辑（仅编辑/仅预览，可切换） |
| 基础格式 | 标题（H1-H6）、加粗、斜体、下划线、删除线、引用、代码块（行内/块级，语法高亮）、有序列表、无序列表、分割线 |
| 高级格式 | 链接（自定义文本+地址）、图片（本地/URL，支持大小调整）、表格（自定义行列、编辑内容、合并/拆分单元格） |
| 自动保存 | 每30秒自动保存一次 |
| 手动保存 | 点击保存按钮 |
| 撤销重做 | Ctrl+Z撤销、Ctrl+Y重做 |
| 语法提示 | 错误MD语法红色下划线+悬浮显示原因 |

### F-MD-003 MD保存与导出

| 需求项 | 内容 |
|--------|------|
| 需求名称 | MD保存导出 |
| 需求描述 | 支持保存和多种格式导出 |
| 手动保存 | 更新文件内容、修改时间、MD5值 |
| 自动保存 | 每30秒，显示"已自动保存"提示 |
| 导出格式 | .md（纯MD）、.pdf、.html |
| 导出样式 | 保持原有格式化样式 |

### F-MD-004 MD预览与分享

| 需求项 | 内容 |
|--------|------|
| 需求名称 | MD预览分享 |
| 需求描述 | 实时预览、临时分享链接 |
| 实时预览 | 右侧预览区实时展示编辑效果 |
| 单独预览 | 脱离编辑模式，展示完整格式化内容 |
| 预览操作 | 放大、缩小、复制文本 |
| 分享链接 | 临时链接，他人可查看（不可编辑） |
| 链接有效期 | 12小时、24小时、7天（可选） |

#### 关键组件

| 组件 | 描述 | 事件 |
|------|------|------|

| `ShareDialog` | 生成分享链接弹窗 | `generate`, `copy` |

#### 交互流程

1. **创建MD流程**:
   ```
   新建文件 → 选择"Markdown" → 输入文件名
   → 自动进入编辑器 → 默认分屏模式
   ```

2. **编辑保存流程**:
   ```
   输入内容 → 30秒debounce → 自动保存
   → 提示"已自动保存" → 手动点击保存按钮 → 提示"保存成功"
   ```

3. **导出流程**:
   ```
   点击导出 → 选择格式 → 点击导出 → 浏览器下载
   ```

4. **撤销重做流程**:
   ```
   Ctrl+Z → 撤销上一步 → 提示"已撤销"
   Ctrl+Y → 重做 → 提示"已重做"
   ```

### 数据结构

**MD内容表 (t_md_content)**

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| file_id | BIGINT | PK, FK→t_file(id) | 文件ID |
| content | TEXT | NOT NULL | MD内容 |
| updated_at | DATETIME | NOT NULL | 更新时间 |

### 接口设计

| 接口 | 方法 | 路径 | 功能 |
|------|------|------|------|
| 创建MD | POST | /api/md/create | 创建MD文件 |
| 获取内容 | GET | /api/md/{fileId}/content | 获取MD内容 |
| 保存内容 | PUT | /api/md/{fileId}/content | 保存MD内容 |
| 导出MD | GET | /api/md/{fileId}/export | 导出MD文件 |
| 生成分享 | GET | /api/md/{fileId}/share | 生成分享链接 |

### 业务规则

1. 文件名：1-50字符，后缀.md
2. 同一目录下文件名唯一
3. 自动保存间隔30秒
4. 导出格式：.md、.pdf、.html
5. 分享链接一次性验证

### 依赖技术

- EasyMDE - Markdown编辑器
- marked.js - Markdown解析
- html2pdf.js - 导出PDF
- highlight.js - 代码高亮
