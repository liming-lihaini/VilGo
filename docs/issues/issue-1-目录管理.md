# ISSUE-目录管理

## 概述

实现目录管理模块的完整功能，包括目录创建、编辑、删除、查询、移动等操作。

## 功能需求

### F-DIR-001 创建目录

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 创建目录 |
| 需求描述 | 用户可在指定父目录下创建新目录 |
| 输入 | 目录名称（1-50字符，无特殊字符） |
| 校验 | 同一父目录下名称唯一 |
| 输出 | 创建成功提示，新目录显示在列表中 |
| 异常处理 | 名称重复提示"重试"，非法字符提示"规则" |
| 技术说明 | 特殊字符：/ \ : * ? " < > \| |

### F-DIR-002 编辑目录

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 编辑目录 |
| 需求描述 | 用户可修改目录名称、移动目录位置 |
| 输入 | 新目录名称（可选）、目标父目录（可选） |
| 校验 | 名称唯一性、目标目录存在 |
| 输出 | 编辑成功提示，目录层级更新 |
| 异常处理 | 目标目录不存在提示错误 |
| 日志记录 | 操作人、操作时间、操作类型、修改前后内容 |

### F-DIR-003 删除目录

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 删除目录 |
| 需求描述 | 用户可删除目录及其下属内容，移入回收站 |
| 输入 | 删除确认 |
| 校验 | 用户有权限操作 |
| 输出 | 确认弹窗，成功后目录从列表移除 |
| 回收站规则 | 保留30天后自动清除 |
| 批量删除 | 支持批量删除，需确认提示 |

### F-DIR-004 目录查询与导航

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 目录查询与导航 |
| 需求描述 | 支持目录搜索、面包屑导航、列表排序 |
| 输入 | 搜索关键词、排序方式 |
| 输出 | 搜索结果列表、目录列表 |
| 排序支持 | 名称、时间升序/降序 |
| 显示信息 | 目录名称、创建时间、创建人 |

## 技术方案

```
┌────────────────────────────────────────────────────────────┐
│  顶部工具栏: [新建目录] [上传文件] [搜索框] [回收站] [设置]   │
├──────────────┬─────────────────────────────────────────────┤
│              │  面包屑: 根目录 > 文档 > 技术资料             │
│   目录树    ├─────────────────────────────────────────────┤
│   侧边栏    │                                             │
│   (20%)     │         目录/文件列表区 (80%)               │
│              │                                             │
│   - 根目录   │  [目录] [文件]  [排序▼]                    │
│     - 文档   │  ┌────┐ ┌────┐ ┌────┐ ┌────┐            │
```

#### 关键组件

| 组件 | 描述 | 事件 |
|------|------|------|
| `DirectoryTree` | 左侧目录树组件，支持展开/折叠/选中 | `select`, `expand`, `refresh` |
| `DirectoryList` | 目录文件列表，表格形式展示 | `click`, `dblclick`, `select` |
| `Breadcrumb` | 面包屑导航，点击跳转 | `click` |
| `Toolbar` | 顶部工具栏，新建/上传等操作 | `click` |
| `ContextMenu` | 右键菜单（重命名/移动/删除） | `select` |
| `CreateDirDialog` | 创建目录弹窗 | `confirm`, `cancel` |
| `ConfirmDialog` | 确认删除弹窗 | `confirm`, `cancel` |

#### 交互流程

1. **创建目录流程**:
   ```
   点击"新建目录" → 弹出创建对话框 → 输入目录名 → 点击确定
   → 调用POST /api/directory/create → 成功→刷新列表 → 失败→提示错误
   ```

2. **删除目录流程**:
   ```
   右键选择"删除" → 弹出确认对话框 → 点击确认
   → 调用DELETE /api/directory/delete → 刷新列表 → 移入回收站
   ```

3. **移动目录流程**:
   ```
   右键选择"移动" → 弹出目录树选择框 → 选择目标目录 → 点击确定
   → 调用PUT /api/directory/update → 刷新列表 → 目录层级更新
   ```

#### 异常提示

| 场景 | 提示信息 |
|------|----------|
| 目录名重名 | "当前目录下已存在同名目录，请修改后重试" |
| 使用特殊字符 | "目录名不能包含以下字符: / \ : * ? \" < > \|" |
| 目录名为空 | "请输入目录名称" |
| 目录名过长 | "目录名称不能超过50个字符" |
| 目标目录不存在 | "目标目录不存在或已删除" |

### 数据结构

**目录表 (t_directory)**

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK, AUTO_INCREMENT | 目录ID |
| name | VARCHAR(50) | NOT NULL | 目录名称 |
| parent_id | BIGINT | FK→t_directory(id) | 父目录ID |
| user_id | BIGINT | FK→t_user(id) | 用户ID |
| path | VARCHAR(500) | - | 完整路径 |
| created_at | DATETIME | NOT NULL | 创建时间 |
| updated_at | NOT NULL | 更新 时间 |
| deleted | TINYINT | DEFAULT 0 | 逻辑删除 |
| delete_at | DATETIME | - | 删除时间(回收站) |

### 索引

```sql
INDEX idx_parent_id (parent_id)
INDEX idx_user_id (user_id)
```

### 接口设计

| 接口 | 方法 | 路径 | 功能 |
|------|------|------|------|
| 获取根目录 | GET | /api/directory/root | 获取根目录 |
| 获取子目录 | GET | /api/directory/children/{parentId} | 获取子目录列表 |
| 创建目录 | POST | /api/directory/create | 创建目录 |
| 更新目录 | PUT | /api/directory/update | 更新目录 |
| 删除目录 | DELETE | /api/directory/delete/{id} | 删除目录 |
| 恢复目录 | POST | /api/directory/restore/{id} | 恢复目录 |
| 搜索目录 | GET | /api/directory/search | 搜索目录 |
| 目录树 | GET | /api/directory/tree | 目录树 |
| 面包屑 | GET | /api/directory/breadcrumb/{id} | 面包屑导航 |

### 业务规则

1. 目录名称：1-50字符，禁止特殊字符 `/ \ : * ? " < > |`
2. 同一父目录下目录名唯一
3. 删除目录时，其下所有子目录和文件一并移入回收站
4. 回收站保留30天后自动永久删除

## 验收条件

1. **创建目录成功** → 目录出现在列表中
2. **重名创建** → 提示"重名"错误
3. **使用特殊字符** → 提示"禁止使用"错误
4. **删除目录** → 弹窗确认 → 目录移入回收站
5. **恢复目录** → 目录恢复正常可见
6. **搜索目录** → 实时返回搜索结果
7. **面包屑导航** → 点击可跳转
8. **排序功能** → 名称/时间升序/降序正常
