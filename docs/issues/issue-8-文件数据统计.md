# ISSUE-文件数据统计

## 概述
在首页中增加文件数据统计视图，展示用户文件的整体数据概况，包括文件数量、目录数量、存储空间使用、文件类型分布、最近修改文件等统计数据，帮助用户快速了解文件管理系统的使用状况。

## 功能需求

### F-STATS-001 基础统计面板
| 需求项 | 内容 |
|--------|------|
| 需求名称 | 基础统计面板 |
| 需求描述 | 在首页展示文件管理的关键统计数据 |
| 统计维度 | 文件总数、目录总数、文件总大小、回收站文件数 |
| 更新策略 | 实时计算/定时刷新 |

### F-STATS-002 文件类型分布
| 需求项 | 内容 |
|--------|------|
| 需求名称 | 文件类型分布 |
| 需求描述 | 统计各类文件的数量分布 |
| 类型分类 | 文档(DOC/DOCX)、表格(XLS/XLSX)、演示(PPT/PPTX)、图片(JPG/PNG/GIF)、PDF、MD、其他 |
| 展示方式 | 饼图/条形图 |

### F-STATS-003 最近操作记录
| 需求项 | 内容 |
|--------|------|
| 需求名称 | 最近操作记录 |
| 需求描述 | 展示最近修改/创建的文件列表 |
| 显示数量 | 最近 10 个文件 |
| 信息项 | 文件名称、修改时间、文件大小 |

### F-STATS-004 存储空间统计
| 需求项 | 内容 |
|--------|------|
| 需求名称 | 存储空间统计 |
| 需求描述 | 展示存储空间使用情况 |
| 统计项 | 已使用空间、可用药用空间、使用占比 |
| 展示方式 | 进度条 + 数值 |

#### 核心组件
| 组件 | 描述 | 事件 |
|------|------|------|
| `StatsPanel` | 统计面板卡片 | `refresh` |
| `FileTypeChart` | 文件类型分布图表 | `click-type` |
| `StorageProgress` | 存储空间进度条 | - |
| `RecentFiles` | 最近操作文件列表 | `open-file` |

### 后端接口设计

#### 统计数据接口
| 接口 | 方法 | 描述 |
|------|------|------|
| `/api/stats/summary` | GET | 获取基础统计数据 |
| `/api/stats/file-types` | GET | 获取文件类型分布 |
| `/api/stats/recent` | GET | 获取最近操作文件 |

### 数据模型

#### StatsSummary
```java
public class StatsSummary {
    private Long directoryCount;    // 目录总数
    private Long fileCount;         // 文件总数
    private Long totalSize;          // 文件总大小(字节)
    private Long trashCount;         // 回收站文件数
    private Long usedStorage;       // 已使用空间
    private Long totalStorage;     // 总存储空间
    private LocalDateTime lastUpdated; // 最后更新时间
}
```

#### FileTypeStat
```java
public class FileTypeStat {
    private String fileType;      // 文件类型
    private Long count;         // 文件数量
    private Long totalSize;    // 该类型总大小
    private Double percentage;  // 占比
}
```

## 验收条件

| 条件 | 描述 | 验证方式 |
|------|------|----------|
| 1. 基础统计 | 首页显示目录数、文件数、存储空间 | 视觉验证 |
| 2. 文件类型分布 | 饼图展示7种类型分布 | 图表验证 |
| 3. 存储进度条 | 显示已用/，可用空间占比 | 进度条验证 |
| 4. 最近操作 | 显示最近10个文件 | 列表验证 |
| 5. 实时更新 | 点击刷新按钮更新统计 | 交互验证 |
| 6. 空状态处理 | 无文件时显示占位符 | 空状态验证 |
| 7. 响应式 | 不同屏幕宽度适配 | 屏幕测试 |
| 8. 加载状态 | 统计加载中显示骨架屏 | 加载验证 |

## Boundaries

### 范围之内
| 类别 | 内容 |
|------|------|
| 后端 | StatsService, StatsController, 统计接口 |
| 前端 | StatsPanel, FileTypeChart, StorageProgress, RecentFiles 组件 |
| 数据 | 文件表、目录表统计查询 |

### 范围之外
| 类别 | 内容 |
|------|------|
| 认证 | 用户登录/权限验证（MVP降级） |
| 图表库 | 不引入新库，使用 Element Plus |
| 导出 | 统计报表导出 |
| 定时 | 服务端定时统计任务 |
