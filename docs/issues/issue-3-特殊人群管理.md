# ISSUE-3-特殊人群管理

## 概述

专门登记脱贫户、监测户、残疾人、孤寡老人等特殊人群信息，可从人事档案模块关联人员基础信息；记录帮扶责任人（关联两委班子成员）、帮扶措施、帮扶时间、帮扶成效，生成帮扶台账；支持帮扶任务提醒（到期自动提示），实现帮扶工作全流程留痕；支持按人群类型、帮扶状态统计，导出帮扶报表。

### 业务规则

1. 人群类型：脱贫户/监测户/残疾人/孤寡老人/其他
2. 帮扶状态：进行中/已完成
3. 帮扶责任人关联两委班子成员（ISSUE-4）
4. 帮扶时间到期前5天自动提醒
5. 身份证号唯一性校验
