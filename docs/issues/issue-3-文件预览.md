# ISSUE-文件预览

## 概述

实现文件预览模块，支持文档、图片、音频、视频的在线预览。

## 功能需求

### F-PREV-001 文档预览

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 文档预览 |
| 需求描述 | 支持PDF/TXT/MD直接预览，Office文件转换后预览 |
| 直接预览 | pdf、txt、md |
| 转换预览 | doc、docx、xls、xlsx、ppt、pptx（转换为PDF） |
| 转换要求 | 单个≤20MB文件转换≤5秒 |
| 预览方式 | 弹出窗口（全屏/弹窗模式） |
| 大文件处理 | ≥20MB显示加载进度 |
| 预览操作 | 切换编辑模式（如MD文件）、下载、删除 |

### F-PREV-002 图片预览

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 图片预览 |
| 需求描述 | 支持图片查看、缩放、旋转、切换 |
| 支持格式 | jpg、jpeg、png、gif、bmp |
| 操作 | 放大、缩小、旋转、上一张、下一张 |
| 预览方式 | 弹出窗口 |

### F-PREV-003 音频预览

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 音频预览 |
| 需求描述 | 支持在线播放 |
| 支持格式 | mp3、wav |
| 操作 | 播放、暂停、音量调节、进度拖拽 |
| 预览方式 | 弹出窗口 |

### F-PREV-004 视频预览

| 需求项 | 内容 |
|--------|------|
| 需求名称 | 视频预览 |
| 需求描述 | 支持在线播放 |
| 支持格式 | mp4、avi |
| 操作 | 播放、暂停、音量调节、进度拖拽、全屏播放 |
| 自适应 | 自适应播放窗口大小 |
| 大文件处理 | ≥20MB显示加载进度 |
| 预览方式 | 弹出窗口 |

#### 关键组件

| 组件 | 描述 | 事件 |
|------|------|------|
| `PreviewDialog` | 通用预览弹窗组件 | `close`, `fullscreen`, `download` |
| `PdfViewer` | PDF预览组件 | `zoom`, `pageChange` |
| `ImageViewer` | 图片预览组件 | `zoom`, `rotate`, `prev`, `next` |
| `MediaPlayer` | 音视频播放组件 | `play`, `pause`, `seek`, `volume` |
| `ConvertProgress` | 文档转换进度条 | `cancel`, `complete` |
| `Thumbnails` | 视频缩略图预览 | `select` |

#### 交互流程

1. **文档预览流程**:
   ```
   双击文件 → 判断文件类型 → 直接预览(pdf/txt/md) 
   → 转换中弹窗(docx等) → 转换完成 → 显示内容
   ```

2. **图片预览流程**:
   ```
   双击图片 → 弹出预览弹窗 → 加载当前图片
   → 点击左右箭头 → 切换上/下一张
   → 点击缩放按钮 → 放大/缩小/旋转
   ```

3. **大文件加载流程**:
   ```
   点击大文件 → 加载进度条 → 逐步加载内容
   → 显示加载百分比 → 加载完成 → 展示内容
   ```

#### 异常提示

| 场景 | 提示信息 |
|------|----------|
| 转换失败 | "文档转换失败，请尝试下载后查看" |
| 加载超时 | "文件加载较慢，是否尝试下载到本地查看？" |
| 格式不支持 | "该文件格式暂不支持预览" |
| 网络错误 | "预览加载失败，请检查网络后重试" |

### 预览服务架构

```
PreviewService
├── getPreviewUrl(fileId) - 获取预览URL
├── convertToPdf(fileId) - Office转PDF
├── generateThumb(fileId) - 生成缩略图
└── getMediaStream(fileId) - 获取媒体流
```

### 接口设计

| 接口 | 方法 | 路径 | 功能 |
|------|------|------|------|
| 预览信息 | GET | /api/preview/{fileId} | 获取文件预览信息 |
| 预览内容 | GET | /api/preview/content/{fileId} | 获取预览内容 |
| PDF预览 | GET | /api/preview/pdf/{fileId} | PDF文件预览 |
| 文本预览 | GET | /api/preview/text/{fileId} | 文本文件预览 |
| HTML预览 | GET | /api/preview/html/{fileId} | HTML预览 |
| 缩略图 | GET | /api/preview/thumb/{fileId} | 获取缩略图 |
| 大图预览 | GET | /api/preview/large/{fileId} | 大图加载 |
| 批量预览 | GET | /api/preview/batch | 批量获取预览信息 |

### 业务规则

1. ≤20MB文件转换时间≤5秒
2. ≥20MB显示加载进度条
3. 转换文件存储在临时目录
4. 转换后文件及时清理

### 依赖技术

- PDF.js - PDF预览
- HTML5 Video/Audio - 音视频播放
- LibreOffice/POI - Office转PDF转换
