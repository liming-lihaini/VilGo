# ISSUE-9-数字大屏

## 概述

可视化展示村级核心数据，数据与各模块实时同步，无需手动更新；支持大屏全屏展示，适配村级办公场所显示设备；展示内容简洁直观，突出核心指标，为村级决策提供直观数据支撑；支持按需求筛选展示内容，简化操作。

### 接口设计

| 接口 | 方法 | 路径 | 功能 |
|------|------|------|------|
| 获取大屏数据 | GET | /api/screen/dashboard | 获取大屏数据 |
| 获取人口数据 | GET | /api/screen/population | 获取人口数据 |
| 获取户籍数据 | GET | /api/screen/household | 获取户籍数据 |
| 获取帮扶数据 | GET | /api/screen/specialGroup | 获取帮扶数据 |
| 获取任务数据 | GET | /api/screen/task | 获取任务数据 |
| 获取活动数据 | GET | /api/screen/activity | 获取活动数据 |

### 业务规则

1. 数据实时聚合，无需手动更新
2. 自动刷新间隔：30秒
3. 全屏模式适配常见分辨率
4. 数据来源各模块实时查询
5. 异常数据友好展示（如数据为空显示"--"）
